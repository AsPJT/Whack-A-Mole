# 導入手順

### 使うもの

|名前|個数|
|:---|:---|
|Processingが実行可能なコンピュータ|1個|
|Arduino Uno|1個|
|3軸加速度センサ（デジタル；MMA8452Q）|1個|
|スピーカ（SP-281W）|1個|
|ブレッドボード|1個|
|配線|数本|
|ハンマー|1個
